Undercurrents

Controls

  • press up / down to modify particle speed
  • press left / right to modify particle line distance factor
  • press b to toggle blank mode
  • press c to clear all particles
  • press f to toggle fading mode
  • press l to toggle particle lines mode
  • press m to toggle color modes
  • press p to pause or unpause visuals
  • press r to randomize colors

More

  • Debug Mode - Run with the console
  • Source Code - on GitHub
  • YouTube Version - With my own music

Notes

This code relies on preserving the drawing buffer for the "fading" effect to work properly which currently is not working in the web/wasm version of this. This means that toggling fade mode with f may not work as expected in this web version. To see the full effect check out the YouTube version above or build this code for your desktop.
